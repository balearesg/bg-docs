<Info>
  ChartComponent es un componente React que utiliza la librería Chart.js para
  generar gráficos personalizados en un canvas{" "}
</Info>

### Props

- chartData: Objeto con los datos que se van a graficar. Este objeto debe tener la siguiente estructura:

- type (opcional): Tipo de gráfico que se quiere generar. Los valores posibles son: line, bar, radar, doughnut, pie, polarArea, bubble, scatter, area. El valor por defecto es doughnut.

### Uso

Para utilizar el componente `ChartComponent`, sigue los siguientes pasos:

1. Importa el componente en tu archivo de React:

<Code>{`import { ChartComponent } from './ChartComponent';`}</Code>

2. Crea un objeto con los datos que quieres graficar, este objeto debe tener la siguiente estructura:

<Code>
  {`
const data = {
  labels: ['Enero', 'Febrero', 'Marzo', 'Abril', 'Mayo', 'Junio'],
  datasets: [
    {
      label: 'Ventas',
      data: [12, 19, 3, 5, 2, 3],
      fill: false,
      borderColor: 'rgb(75, 192, 192)',
      tension: 0.1,
    },
  ],
};
`}
</Code>

3. Utiliza el componente ChartComponent en tu renderizado de React:

<Code>
  {`
function App() {
  return (
    <div>
      <h1>Mi gráfico</h1>
      <ChartComponent chartData={data} type="line" />
    </div>
  );
}
`}
</Code>
