<Title text="Componentes a ser usados al documetar" />

<Info>
  En este documento se explica como importar componentes hechos sobre React en
  archivos .mdx así como también que componentes se deben usar para casos
  específicos y como usarlos; Para poder usar estos components en un archivo
  .mdx deben ser importados en dicho archivo.
  <strong>
    importante dejar una linea de separación entre las importaciones y la
    implementación
  </strong>
</Info>

<Title tag="h3" text="Componente Title ( Título )" />

Componente para ser usado en los títulos y subtítulos de los documentos, para
importarlo:

<Code>{`import { Title } from "@bg/docs/components/html";`}</Code>

Para usarlo como título:

<Code>{`<Title text="Esto es un titulo" />`}</Code>

Resultado:

<Title text="Esto es un titulo" />

Para usarlo como subtítulo:

<Code>{`<Title tag="h3" text="Esto es un subtitulo" />`}</Code>

Resultado:

<Title tag="h3" text="Esto es un subtitulo" />

<Title tag="h3" text="Componente Link" />

Componente para ser usado cuando se quiera o necesite colocar un link, para
importarlo:

<Code>{`import { Link } from "@bg/docs/components/html";`}</Code>

Para usarlo

<Code>{`<Link href="https://docs.balearesgroup.com/">Esto es un link<Link />`}</Code>

Resultado: 

<Link href="https://docs.balearesgroup.com/">Esto es un link</Link >
