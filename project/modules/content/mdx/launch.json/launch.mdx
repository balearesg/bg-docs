# Configuración de `launch.json` en VSCode para Proyectos BeyondJS

## Creación o Modificación del archivo `launch.json`

### Abrir VSCode

1. Asegúrate de que tu proyecto BeyondJS esté abierto en VSCode.

### Acceso a Configuraciones de Depuración

1. En la barra lateral izquierda de VSCode, encontrarás el icono de depuración (parece una mariquita). Haz clic en él.
2. En la parte superior, verás un dropdown y un botón de configuración (parece un engranaje). Haz clic en el botón de
   configuración.

### Creación/Edición del archivo `launch.json`

1. Si tu proyecto no tiene un archivo `launch.json`, VSCode te pedirá que lo crees. Selecciona el entorno "Node.js".
2. Si ya tienes un archivo `launch.json`, se abrirá. Si no, VSCode creará uno nuevo con algunas configuraciones
   predeterminadas.

### Ingresar Configuración

Reemplaza el contenido existente o agrega la configuración proporcionada en tu archivo `launch.json`.

```json
{
	"version": "0.2.0",
	"configurations": [
		{
			"type": "node",
			"request": "launch",
			"name": "Beyond",
			"runtimeExecutable": "beyond",
			"args": ["run"]
		}
	]
}
```

## Entendiendo la Configuración

-   `version`: Este campo especifica la versión del formato de configuración. Siempre es "0.2.0".
-   `configurations`: Es un arreglo de configuraciones de depuración. Cada configuración es un objeto con varios campos.
    -   `type`: Especifica el tipo de depurador a usar, en este caso, es "node" para Node.js.
    -   `request`: Define cómo se iniciará la depuración. "launch" indica que se iniciará una nueva instancia del
        programa.
    -   `name`: Es el nombre de esta configuración de depuración.
    -   `runtimeExecutable`: Especifica el ejecutable del tiempo de ejecución que se utilizará.
    -   `args`: Es un arreglo de argumentos de línea de comandos que se pasarán al programa.

## Ejecutando la Depuración

1. **Selecciona la Configuración**: Asegúrate de que "Beyond" esté seleccionado en el dropdown de configuración de
   depuración.
2. **Inicia la Depuración**: Haz clic en el botón verde de "Iniciar Depuración" (parece un triángulo verde) para iniciar
   la depuración de tu proyecto.
